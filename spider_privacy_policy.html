Política de Privacidade - Spider Pro - Card Challenge

Última atualização: 27 de outubro de 2025

Resumo: O Spider Pro - Card Challenge respeita sua privacidade. Este jogo coleta dados mínimos necessários para funcionalidade básica e publicidade. Não vendemos seus dados pessoais.

1. Introdução

Bem-vindo ao Spider Pro - Card Challenge ("nós", "nosso" ou "Jogo"). Esta Política de Privacidade explica como coletamos, usamos, divulgamos e protegemos suas informações quando você joga nosso jogo de paciência Spider.

Ao usar o Spider Pro - Card Challenge, você concorda com a coleta e uso de informações de acordo com esta política.

2. Informações que Coletamos

2.1. Dados de Jogo

  • Progresso do Jogo: Pontuações, estatísticas de vitórias/derrotas, níveis completados, recordes pessoais
  • Configurações: Preferências de dificuldade (1 naipe, 2 naipes, 4 naipes), som, temas visuais
  • Dados salvos localmente: Todo o progresso do jogo é armazenado localmente no seu dispositivo

2.2. Informações de Dispositivo

  • Modelo do dispositivo, versão do sistema operacional Android
  • Identificador de publicidade do Google (GAID)
  • Informações de crash e desempenho do aplicativo
  • Endereço IP e informações de conexão

2.3. Dados Analíticos

  • Tempo de jogo, sessões iniciadas
  • Interações com funcionalidades do jogo
  • Dados de desempenho do aplicativo

3. Como Usamos Suas Informações

Utilizamos as informações coletadas para:

  • Funcionalidade do Jogo: Salvar seu progresso, manter estatísticas e recordes
  • Melhorias: Analisar padrões de uso para melhorar a jogabilidade e corrigir bugs
  • Publicidade: Exibir anúncios relevantes através do Google AdMob
  • Segurança: Detectar e prevenir fraudes ou atividades maliciosas
  • Suporte: Responder a solicitações de suporte técnico

4. Serviços de Terceiros

O Spider Pro - Card Challenge integra os seguintes serviços de terceiros:

4.1. Google AdMob

  • Exibe anúncios dentro do jogo
  • Coleta identificadores de publicidade e dados de interação com anúncios
  • Pode usar cookies e tecnologias similares para publicidade personalizada

Política de Privacidade do AdMob: https://policies.google.com/privacy

Gerenciar preferências de anúncios: https://adssettings.google.com

4.2. Google Play Services

  • Gerenciamento de compras in-app (versão sem anúncios)
  • Serviços de análise e estatísticas

4.3. Firebase Analytics (Google)

  • Coleta dados de uso anônimos para análise de desempenho
  • Informações sobre comportamento no aplicativo

Política de Privacidade do Firebase: https://firebase.google.com/support/privacy

5. Compartilhamento de Dados

Nós NÃO vendemos seus dados pessoais.

Compartilhamos informações apenas nas seguintes circunstâncias:

  • Provedores de Serviço: Com Google AdMob e Firebase para funcionalidade de anúncios e análises
  • Requisitos Legais: Quando exigido por lei, ordem judicial ou processo legal
  • Proteção de Direitos: Para proteger nossos direitos, propriedade ou segurança

6. Armazenamento de Dados

  • Dados Locais: Seu progresso e configurações do jogo são armazenados localmente no seu dispositivo
  • Dados Analíticos: Dados anônimos coletados por Firebase e AdMob são retidos de acordo com as políticas dessas plataformas
  • Exclusão: Ao desinstalar o jogo, todos os dados locais são removidos. Dados analíticos anônimos podem permanecer nos serviços de terceiros

7. Segurança dos Dados

Implementamos medidas de segurança para proteger suas informações:

  • Armazenamento seguro de dados locais no dispositivo
  • Comunicação criptografada com servidores de terceiros (HTTPS)
  • Uso de identificadores anônimos quando possível

Importante: Nenhum método de armazenamento ou transmissão é 100% seguro. Embora nos esforcemos para proteger seus dados, não podemos garantir segurança absoluta.

8. Seus Direitos e Escolhas

8.1. Controle de Anúncios

  • Desativar anúncios personalizados nas configurações de anúncios do Google
  • Adquirir a versão premium sem anúncios (compra in-app)
  • Usar as configurações de privacidade do seu dispositivo Android para limitar rastreamento

8.2. Exclusão de Dados

  • Dados do Jogo: Desinstalar o aplicativo remove todos os dados locais
  • Dados Analíticos: Para solicitar exclusão de dados analíticos, entre em contato conosco

8.3. Permissões do Aplicativo

O Spider Pro - Card Challenge solicita as seguintes permissões:

  • Armazenamento: Para salvar seu progresso e configurações
  • Internet: Para exibir anúncios e coletar análises anônimas
  • Acesso ao estado da rede: Para verificar conectividade antes de carregar anúncios

9. Privacidade de Crianças

O Spider Pro - Card Challenge é adequado para todas as idades. No entanto, não coletamos intencionalmente informações pessoais identificáveis de crianças menores de 13 anos.

Se você é pai ou responsável e acredita que seu filho forneceu informações pessoais, entre em contato conosco para que possamos remover essas informações.

10. Transferências Internacionais de Dados

Seus dados podem ser processados em servidores localizados fora do seu país, incluindo nos Estados Unidos (onde Google/Firebase mantêm servidores).

Ao usar o Spider Pro - Card Challenge, você consente com essa transferência. Utilizamos provedores que aderem a padrões internacionais de proteção de dados.

11. Alterações a Esta Política

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas:

  • Publicando a nova política nesta página
  • Atualizando a data de "Última atualização"
  • Exibindo notificação no jogo (para mudanças importantes)

Recomendamos revisar esta política periodicamente.

12. Lei Geral de Proteção de Dados (LGPD) - Brasil

Para usuários no Brasil, garantimos seus direitos sob a LGPD:

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados coletados
  • Correção de dados incompletos ou inexatos
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados (quando aplicável)
  • Informação sobre compartilhamento de dados
  • Revogação do consentimento

13. Regulamento Geral de Proteção de Dados (GDPR) - União Europeia

Para usuários na União Europeia, garantimos seus direitos sob o GDPR:

  • Direito de acesso aos seus dados pessoais
  • Direito de retificação de dados imprecisos
  • Direito ao esquecimento (exclusão de dados)
  • Direito de restrição de processamento
  • Direito à portabilidade de dados
  • Direito de objeção ao processamento
